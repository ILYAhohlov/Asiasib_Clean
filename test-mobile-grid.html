Инструкции для тестирования:

  1. Откройте этот файл в браузере
  2. Откройте инструменты разработчика (F12)
  3. Переключитесь в режим мобильного устройства
  4. Проверьте, какая сетка отображается правильно в 2 колонки
  5. Проверьте на разных размерах экрана
